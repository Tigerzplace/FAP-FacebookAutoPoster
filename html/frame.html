What is Facebook Auto Poster?

Facebook Auto Poster is a Chrome extension that automates posting to multiple Facebook groups with one click, helping you save time and improve engagement.

This tool also works with new version of Facebook.
It is not recommended to post more than 60 requests per posting with less wait time.
